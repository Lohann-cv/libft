### Description
The goals of this project were to creat your own library with some of the basic manual functions
As well as a make file to compile all the files
### Instructions
To creat the library and clean all the files you must :
1. Do the "make" command.
2. The archive should be created
3. Then do the "make fclean" command to clean all the files
